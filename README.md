# FlowPilot AI

AI 기반 서비스 플로우 설계 툴 프로토타입입니다. 자연어 입력으로 프로젝트를 시작하고, 상단 캔버스에서 플로우차트를 확인하며, 하단 AI Co-pilot 패널에서 논리 결함, 영향도, 인프라 체크리스트를 검토하는 구조입니다.

## 실행

의존성 설치가 필요 없는 정적 웹앱입니다.

```bash
node server.mjs
```

브라우저에서 `http://127.0.0.1:5173/`를 엽니다.

또는 `flowpilot-ai/index.html`을 직접 열어도 동작합니다.

## 주요 기능

- 자연어 입력 기반 프로젝트 생성 화면
- 상단 60% 플로우 캔버스, 하단 40% AI 채팅/컨트롤
- 순차 노드 생성 애니메이션
- 노드 드래그 이동 및 edge 재계산
- 노드 더블클릭 삭제 fade-out 모션
- AI Co-pilot 질문, 선택지, Infra-Checker 체크리스트
- Impact Analysis 오버레이
- Export 모달

## 디자인 방향

Voltagent 디자인 언어를 라이트 모드 B2B SaaS 제품 UI로 재해석했습니다.

- Inter 기반의 차분한 타이포그래피
- 전기 녹색 `#00d992`를 핵심 액션과 정상 상태에만 사용
- 얇은 hairline border 중심의 패널/노드 구조
- SF Mono 계열 메타 라벨로 기술적 신뢰감 부여
- 주니어에게는 명확한 선택지를, 시니어에게는 영향도와 리스크 정보를 빠르게 제공
